# GinChat
基于Go语言Gin框架开发的在线客服SAAS系统
## 项目目录结构说明
<pre><code>
├── bootstrap -初始化
├── cmd -命令行
├── common -公共
├── config -配置文件
├── controller -控制器
│── doc -文档
│── log -日志
│── middleware -中间件
│── model -模型
│── route -路由
│── service -服务
│── static -静态资源
│── template -模板
│── test -测试
│── util -通用库
</code></pre>

## 运行方式
<code>
go run main.go
</code>
